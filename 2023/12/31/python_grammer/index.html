Python Grammer

python基础语法

  • 标识符命名规则

    • 由数字、字母、下划线组成
    • 不能以数字开头
    • 不能使用内置关键字
    • 严格区分大小写

  • 命名习惯

    • 见名知义
    • 下划线

  • 检测数据类型的方法：type()

  • 数据类型：整型int、浮点型float、字符串str、布尔型bool、元组tuple、集合set、字典dict

  • 格式符号

  • print('...<格式符号>...' % 变量名)或者printf(f'...{变量名}...')
  • print(end="\n")默认自带换行结束符，可按需求更改，如不换行版本print(end="")
  • 输入：input("提示信息")
    • 会把接受到的任意用户输入的数据都当作字符串处理
  • 转换数据类型的函数

  • 算术运算符

  • 逻辑运算符

  • 运算符的其他用法

  • 常用方法

  • range()生成的序列不包含end数字

  • 容器类型转换
转换为元组 tuple()
转换为列表 list()
转换为集合 set()
  • 条件语句
  • 三元运算符
  • 循环
  • 函数定义
  • 调用函数时根据函数定义的参数位置来传递参数
  • 可通过key = value的形式加以指定（此时清除了参数的顺序需求）
    • 位置参数（若有）必须在关键字参数的前面，关键字参数之间不存在先后顺序
  • 缺省参数：为参数提供默认值，调用时可不指定该默认参数的值
    • 所有位置参数必须出现在默认参数前，包括函数定义和调用
  • 不定长参数
    • 单个*以元组形式合并
    • 两个*以字典形式合并
  • 拆包
    • 对元组拆包得到的是元组中的元素
    • 对字典拆包得到的是字典中的关键字
  • 使用id()来判断两个变量是否为同一个值的引用
  • b = a应理解为b与a指向同一块内存区域
  • 可变类型（数据能够直接修改）：列表、字典、集合
  • 不可变类型（重新赋值表示指向另一块区域）：整型、浮点型、字符串、元组

python容器

  • 列表
[data1, data2, ...] 初始化
list_name[0/1/...] 访问
list_name.index(data, start, end) 返回指定数据所在位置的下标；不存在则报错
list_name.count(data) 返回指定数据在列表中出现的次数
len(list_name) 返回列表长度
list_name.append(data) 追加数据；若为列表则整个列表作为一个元素
list_name.extend(data) 追加数据；若为列表则列表元素逐个添加
list_name.insert(index, data) 指定位置新增数据
del list_name/list_name[index] 删除列表、指定数据
list_name.pop(index) 删除指定下标的数据（默认最后一个），并返回
list_name.remove(data) 删除第一个匹配项
list_name.clear() 清空列表
list_name[index] = data 修改元素
list_name.reverse() 逆置
list_name.sort(key = None, reverse False) 排序，reverse = False表示升序（默认）
list_name.copy() 复制
  • 存在in与不存在not in
  • 列表的遍历
  • 元组
    • 一个元组可以存储多个数据，但元组内的数据不能修改（修改则报错，但可修改内部的列表元素，若有），数据可以是不同类型
    • 一个数据的元组，数据后面最好也添加逗号
(data1, data2, ...) 定义元组
tuple_name[index] 按下标访问
tuple_name.index(data) 查找数据在元组中的下标
tuple_name.count(data) 统计某个数据在元组中出现的次数
len(tuple_name) 元组中数据的个数
  • 字符串
    • 初始化：'abc'/'''可换行'''
    • 可按下标访问
    • 对字符串的修改不会改变原有字符串
str_name.find(str, start, end) 检查某个子串是否在该字符串中，返回该子串开始的位置下标；否则返回-1
rfind() 查找从右侧开始
str_name.index(str, start, end) 检查某个子串是否在该字符串中，返回该子串开始的位置下标；否则报错
rindex() 查找从右侧开始
str_name.count(str, start, end) 返回某个子串在字符串中出现的次数
str_name.replace(old_str, new_str, num) num为替换次数，默认为str_name.count(old_str)
str_name.split(分割字符, num) 按照指定字符分割字符串，num为分割字符出现的次数（即返回的列表的大小为num+1）
str.join(多字符串组成的序列) 用str连接多个字符串，形成一个新的字符串
str_name.capitalize() 将字符串的第一个字符大写
str_name.title() 将字符串每个单词首字母大写
str_name.lower() 将字符串中大写转小写
str_name.upper() 将字符串中小写转大写
str_name.lstrip() 删除字符串左侧空白字符
str_name.rstrip() 删除字符串右侧空白字符
str_name.strip() 删除字符串两侧空白字符
str_name.ljust(length, 填充字符) 返回一个原字符串左对齐，并使用指定字符（默认空格）填充至指定长度的新字符串
str_name.rjust(length, 填充字符) 返回一个原字符串右对齐，并使用指定字符（默认空格）填充至指定长度的新字符串
str_name.center(length, 填充字符) 返回一个原字符串居中对齐，并使用指定字符（默认空格）填充至指定长度的新字符串
str_name.startswith(str, start, end) 检查字符串是否以指定子串开头
str_name.endswith(str, start, end) 检查字符串是否以指定子串结尾
str_name.isalpha() 至少有一个字符且所有字符都是字母
str_name.isdigit() 字符串只包含数字
str_name.isalnum() 字符串至少有一个字符且所有字符都是字母或数字
str_name.isspace() 字符串只包含空白
  • 切片：对操作的对象（列表、元组、字符串）截取其中一部分
    • 语法：sequence[start: end: step]
    • 不包含结束位置下标对应的数据，正负整数均可
    • 步长可正可负（负数表示倒序切片），默认为1
  • 集合
    • 创建集合：{data1, data2, ...}或set(data1, data2, ...)
    • 创建空集合只能用set()（{}表示空字典）
    • 集合可以去掉重复数据
    • 集合数据是无序的，不支持下标访问
set_name.add(data) 增加元素（若已存在则不进行任何操作）
set_name.update(list) 追加序列或字符串（逐个追加）
set_name.remove(data) 删除指定数据（若不存在则报错）
set_name.discard(data) 删除指定数据（若不存在则不会报错）
set_name.pop() 随机删除集合中的某个数据，并返回
  • 字典
    • 元素以键值对形式出现
    • 不支持下标访问，支持键访问
{key1: value1, key2: value2, ...}/{}/dict() 创建字典、创建空字典
dict_name[key] = value 增（不存在key）或改（已存在key）
del dict[key] 删除指定键值对
dict_name.clear() 清空字典
dict_name[key] 按key查找；不存在则报错
dict_name.get(key, default) 按key查找；不存在则返回default，默认为None
dict_name.keys() 返回键的dict_keys对象（内容为键构成的列表）
dict_name.values() 返回值的dict_values对象（内容为值构成的列表）
dict_name.items() 返回键值对的dict_items对象（内容为键值对元组构成的列表）
  • lambda语法：lambda 参数列表 : 表达式
  • 直接打印lambda表达式，输出的是lambda的内存地址
  • lambda的参数形式
  • 高阶函数：将函数作为参数传入
  • 内置高阶函数
    • map(func, lst)：将传入的函数变量func作用到lst变量的每个元素中，并将结果组成新的迭代器返回
    • reduce(func, lst)（func有两个参数，导入functools）：每次func计算的结果继续和序列的下一个元素做累积计算
    • filter(func, lst)：过滤掉不符合条件的元素，返回一个filter对象
  • 推导式举例

python文件操作

  • 文件的打开：file_object = open(file_path, mode)

  • 文件的关闭：file_object.close()
  • 写：file_object.write(content)
  • 读
    • file_object.read(num)：num表示从文件中读取的数据的长度（单位：字节），默认所有数据
    • file_object.readlines()：按照行的方式把整个文件中的内容进行一次性读取，返回一个列表（其中每一行的数据作为一个元素）
    • file_object.readline()：一次读取一行内容
  • file_object.seek(偏移量, 起始位置)：用于移动文件指针
    • 起始位置：0-文件开头；1-当前位置；2-文件结尾
  • 文件和文件夹的操作（预先导入os模块）
os.rename(old_file_name, new_file_name) 文件重命名
os.remove(file_name) 删除文件
os.mkdir(dir_name) 创建文件夹
os.rmdir(dir_name) 删除文件夹
os.getcwd() 获取当前目录（绝对路径）
os.chdir(dir) 改变默认目录
os.listdir(dir) 获取目录列表（dir默认为./）

python模块和包

  • 同名功能，调用的是最后导入的模块的功能
  • 当导入一个模块，Python解析器对模块位置的搜索顺序为
    1. 当前目录
    2. shell变量PYTHONPATH下的每个目录
    3. 默认路径
  • 若一个模块文件有__all__变量（一个以功能名字符串为元素的列表），当使用from module_name import *导入时，只能导入该列表中的元素对应的功能
  • 包
    • 将有联系的模块组织在一起，即放在同一个文件夹下
    • 并且在该文件夹创建一个名为__init__.py的文件

python面向对象

  • self指调用该函数的对象
  • 对象属性既可以在类外部添加和获取，也可以在类内部添加和获取
  • 魔法方法（具有特殊功能的函数）
    • __init__()：初始化对象，创建对象时默认被调用，python解释器自动传递self参数
    • __str__()：设置打印对象的形式（默认打印对象的内存地址）
    • __del__()：删除对象时，默认调用
  • 继承
  • 所有类默认继承object类，object类是顶级类或基类，其他子类称为派生类
  • 多继承：SuperClass之间用逗号隔开（左边优先级更高）
  • SubClass和SuperClass具有同名属性和方法，默认使用SubClass的
  • SubClass中调用SuperClass成员
    • SuperClass.mem_var和SuperClass.mem_func(self)
    • super().mem_var和super().mem_func()（比较适合单继承使用）
  • 设置私有权限的方法：在属性名和方法名前添加__
  • pass表示占位，没有实际意义
  • 一般定义函数get_xxx(self)用于获取私有属性值，定义函数set__xxx(self)来修改私有属性值
  • 类属性：归属于类对象的属性，所有对象共有的属性
