基础选择器

标签选择器

  • 使用标签名作为选择器
  • 选中同名标签设置相同的样式

类选择器

id选择器

  • 同一个id选择器在页面中只能用一次

通配符选择器

  • 选择所有标签设置相同样式
  • 开发中一般用来清除标签的默认样式

画盒子

  • px表示像素

文字控制属性

字体大小

  • 谷歌浏览器字体默认大小为16px

字体粗细

  • 数字或关键字（一般写数字）
    • 正常：400, normal
    • 加粗：700, bold

字体倾斜

  • 常用于清除文字默认的倾斜效果
    • 正常：normal
    • 倾斜：italic

行高

  • 数字px或纯数字（表示当前标签font-size属性值的倍数）
  • 行高 = 文字上间距 + 文字高度 + 文字下间距
  • 行高 = 某行文字的最顶端（最底端）到下一行文字的最顶端（最底端）的距离

文字垂直居中

  • 令line-height: 所属盒子的高度;
  • 只适用于单行文字

字体族

  • 多个字体之间用逗号隔开
  • 浏览器会从前往后直到找到一个可用的字体
  • 最后一个字体一般为一个字体族名
  • 网页开发推荐使用无衬线字体，即sans-serif

字体复合属性

  • 一般用于设置网页文字公共样式
  • font: 是否倾斜 是否加粗 字号/行高 字体;
  • 必须按顺序书写，空格隔开
  • 字号和字体值必须写，否则font属性不生效

文本缩进

  • 数字px或数字em（1em = 当前标签的字号大小）

文本对齐

  • left(default)/center/right
  • 调整的是内容的对齐方式，而不是标签的对齐方式

图片居中

  • <div><img></div>
  • 设置div的text-align属性为center

修饰线

  • none(无), underline(下划线), line-through(删除线), overline(上划线)

颜色

  • 颜色关键字
  • rgb表示法
  • rgba表示法(a表示透明度, 取值为0~1之间的小数, 0表示完全透明, 1表示完全不透明)
  • 十六进制表示法

复合选择器

后代选择器

  • 选中某元素的后代元素
  • 空格隔开即可

子代选择器

  • 选中某元素的子代元素（最近的子级）
  • 大于号隔开即可

并集选择器

  • 选中多组标签设置相同的样式
  • 逗号隔开即可（推荐逗号 + 换行）

交集选择器

  • 选中同时满足多个条件的元素
  • 选择器之间连写即可
  • 标签选择器（若有）应放在最前面

伪类选择器

  • 伪类：元素的状态
  • 选中元素的某个状态设置样式
  • link(访问前), visited(访问后), hover(鼠标悬停), active(点击瞬间)
  • 设置超链接的状态时按照link, visited, hover, active的顺序写
  • 任何标签都可以设置鼠标悬停状态
  • 开发中一般设置默认超链接样式和鼠标悬停超链接样式

CSS特性

继承性

  • 子级默认继承父级的文字控制属性
  • 开发中一般在body中设置文字控制属性, 其他需要改变的单独设置
  • 标签本身的特殊样式不会被继承覆盖，即继承失效（如超链接的蓝色不会被覆盖）

层叠性

  • 相同的属性会覆盖：后面的CSS属性覆盖前面的CSS属性
  • 不同的属性会叠加：不同的CSS属性都生效

优先级

  • 优先级（又称权重）：当一个标签使用了多种选择器时，基于不同种类的选择器的匹配规则
  • 规则：优先级高的样式生效
  • 通配符选择器 < 标签选择器 < 类选择器 < id选择器 < 行内样式 < !important
  • 即选中标签的范围越大，优先级越低
