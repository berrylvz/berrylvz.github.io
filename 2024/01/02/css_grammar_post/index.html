CSS grammar

基础选择器

标签选择器

  • 使用标签名作为选择器
  • 选中同名标签设置相同的样式

类选择器

id选择器

  • 同一个id选择器在页面中只能用一次

通配符选择器

  • 选择所有标签设置相同样式
  • 开发中一般用来清除标签的默认样式

画盒子

  • px表示像素

文字控制属性

字体大小

  • 谷歌浏览器字体默认大小为16px

字体粗细

  • 数字或关键字（一般写数字）
属性值 数字 含义
normal 400 正常
bold 700 加粗

字体倾斜

  • 常用于清除文字默认的倾斜效果
属性值 含义
normal 正常
italic 倾斜

行高

  • 数字px或纯数字（表示当前标签font-size属性值的倍数）
  • 行高 = 文字上间距 + 文字高度 + 文字下间距
  • 行高 = 某行文字的最顶端（最底端）到下一行文字的最顶端（最底端）的距离

文字垂直居中

  • 令line-height: 所属盒子的高度;
  • 只适用于单行文字

字体族

  • 多个字体之间用逗号隔开
  • 浏览器会从前往后直到找到一个可用的字体
  • 最后一个字体一般为一个字体族名
  • 网页开发推荐使用无衬线字体，即sans-serif

字体复合属性

  • 一般用于设置网页文字公共样式
  • font: 是否倾斜 是否加粗 字号/行高 字体;
  • 必须按顺序书写，空格隔开
  • 字号和字体值必须写，否则font属性不生效

文本缩进

  • 数字px或数字em（1em = 当前标签的字号大小）

文本对齐

  • left(default)/center/right
  • 调整的是内容的对齐方式，而不是标签的对齐方式

图片居中

  • <div><img></div>
  • 设置div的text-align属性为center

修饰线

属性值 含义
none 无
underline 下划线
line-through 删除线
overline 上划线

颜色

属性写法 举例 备注
颜色关键字 color: red;
rgb表示法 color: rgb(255, 0, 0) a表示透明度, 取值为0~1之间的小数, 0表示完全透明, 1表示完全不透明)
rgba表示法 color: rgba(255, 0, 0, 1)
十六进制表示法 color: #ff0000

复合选择器

后代选择器

  • 选中某元素的后代元素
  • 空格隔开即可

子代选择器

  • 选中某元素的子代元素（最近的子级）
  • 大于号隔开即可

并集选择器

  • 选中多组标签设置相同的样式
  • 逗号隔开即可（推荐逗号 + 换行）

交集选择器

  • 选中同时满足多个条件的元素
  • 选择器之间连写即可
  • 标签选择器（若有）应放在最前面

伪类选择器

  • 伪类：元素的状态
  • 选中元素的某个状态设置样式
属性值 含义
link 访问前
visited 访问后
hover 鼠标悬停
active 点击瞬间
  • 设置超链接的状态时按照link, visited, hover, active的顺序写
  • 任何标签都可以设置鼠标悬停状态
  • 开发中一般设置默认超链接样式和鼠标悬停超链接样式

CSS特性

继承性

  • 子级默认继承父级的文字控制属性
  • 开发中一般在body中设置文字控制属性, 其他需要改变的单独设置
  • 标签本身的特殊样式不会被继承覆盖，即继承失效（如超链接的蓝色不会被覆盖）

层叠性

  • 相同的属性会覆盖：后面的CSS属性覆盖前面的CSS属性
  • 不同的属性会叠加：不同的CSS属性都生效

优先级

  • 优先级（又称权重）：当一个标签使用了多种选择器时，基于不同种类的选择器的匹配规则
  • 规则：优先级高的样式生效
  • 通配符选择器 < 标签选择器 < 类选择器 < id选择器 < 行内样式 < !important
  • 即选中标签的范围越大，优先级越低
  • !important提高属性的优先级，慎用

优先级叠加计算规则

  • 如果是复合选择器，则需要权重叠加计算
  • 行内样式, id选择器个数, 类选择器个数, 标签选择器个数
  • 从左向右依次比较选择器个数，同一级个数多的优先级高，如果个数相同，则向后比较
  • !important权重最高，继承权重最低

Emmet写法

背景属性

背景色

背景图片

  • 默认平铺

背景图片平铺方式

属性值 效果 备注
no-repeat 不平铺 默认盒子左上角，常用
repeat 平铺 默认效果
repeat-x 水平方向平铺
repeat-y 垂直方向平铺

背景图片位置

  • 属性值：水平方向位置 垂直方向位置
  • 关键字或坐标（数字px，正负均可），可混用
  • 正数向右/下移动，负数向左/上移动
  • 0 0或left top为默认值
关键字 含义
left 左侧
right 右侧
center 居中
top 顶部
bottom 底部
  • 关键字取值写法可以颠倒顺序
  • 只写一个关键字，则另一个方向默认居中
  • 只写一个数字，则默认表示水平位置，垂直方向默认居中

背景图片缩放

属性值写法 含义
关键字
百分比 根据盒子尺寸计算图片大小
数字+单位（如px） 不常用
关键字 含义
cover 等比例缩放图片以完全覆盖背景区，可能背景图片部分看不见
contain 等比例缩放图片以完全放入背景区，可能背景区部分空白（即等比例缩放宽高直至宽或高达到盒子的宽或高）
  • 100%：图片宽度达到盒子宽度，高度等比例缩放

背景图片固定

  • 背景不会随着元素的内容滚动

背景复合属性

  • 背景色 背景图片 背景图片平铺方式 背景图片位置/背景图片缩放 背景图片固定
  • 空格隔开即可，不区分顺序

显示模式

显示模式 占 设置宽高属性 宽高尺寸
块级元素 独占一行 生效 宽度默认是父级的100%
行内元素 一行可以显示多个 不生效 宽高尺寸由内容撑开
行内块元素 一行可以显示多个 生效 宽高尺寸由内容撑开

转换显示模式

属性值 效果
block 块级
inline-block 行内块
inline 行内

选择器

结构伪类选择器

  • 根据元素的结构关系查找元素
选择器 说明
E:first-child 查找第一个E元素
E:last-child 查找最后一个E元素
E:nth-child(N) 查找第N个E元素（第一个元素N为1）
E:nth-child(f(N)) 查找所有第f(N)个E元素（N从0递增，倍数不需要写*）

伪元素选择器

  • 创建虚拟元素（伪元素），用来摆放装饰性的内容
选择器 说明
E::before 在E元素里面的最前面添加一个伪元素
E::after 在E元素里面的最后面添加一个伪元素
  • 必须设置content: "";属性，用来设置伪元素的内容（若没有内容，引号留空即可）
  • 伪元素默认是行内显示模式
  • 权值和标签选择器相同

盒子模型

盒子模型的组成

组成 属性
内容区域 width&height
内边距（出现在内容与盒子边缘之间） padding
边框线 border
外边距（出现在盒子外面） margin

边框线

  • 属性值：边框线粗细 线条样式 颜色（不区分顺序）
属性值 线条样式
solid 实线
dashed 虚线
dotted 点线
  • border-方位名词: 属性值;

内边距

  • 设置内容与盒子边缘之间的距离
  • padding/padding-方向名词: ;
  • 多值写法
取值个数 示例 top right bottom left
一个值 a a a a a
两个值 a b a b a b
三个值 a b c a b c b
四个值 a b c d a b c d
  • 规律：从上开始顺时针赋值（上右下左），当前方向没有数值则与对面取值相同

尺寸计算

  • 盒子尺寸 = 内容尺寸 + border尺寸 + 内边距尺寸
  • 给盒子加border/padding属性会撑大盒子
  • 解决
    • 手动做减法：减去border & padding尺寸
    • 内减模式：box-sizing: border-box;

外边距

  • 拉开两个盒子之间的距离

  • 属性值写法与padding相同

  • 版心居中

    • 设置左右margin值为auto
    • 盒子必须有宽度
  • 外边距不会撑大盒子，故不需要考虑盒子尺寸问题

清除默认样式

  • 清除标签默认的样式

元素溢出

  • 控制溢出元素的内容的显示方式
属性值 效果
hidden 溢出隐藏（最常用）
scroll 溢出滚动（无论是否溢出，都显示滚动条）
auto 溢出才显示滚动条

外边距合并现象

  • 垂直排列的兄弟元素，上下margin会合并
  • 取两个margin中的较大值（上元素的margin-bottom和下元素的margin-top）生效
  • 解决：只给其中一个盒子设置外边距即可

外边距塌陷问题

  • 父子级的标签，子级添加上外边距margin-top会产生塌陷问题
  • 导致父级一起向下移动
  • 解决
    • （推荐）取消子级margin，父级设置padding
      • 但同时会撑大盒子，因此还需设置box-sizing: border-box;
    • 父级设置overflow: hidden;
    • 父级设置border-top

行内元素的垂直位置问题

  • 行内元素添加margin和padding，无法改变元素垂直位置
  • 解决：给行内元素添加line-height属性

圆角

  • 设置元素的外边框为圆角
  • 属性（圆角半径）：数字px/百分比
取值个数 示例 左上 右上 右下 左下
一个值 a a a a a
两个值 a b a b a b
三个值 a b c a b c b
四个值 a b c d a b c d
  • 从左上角开始顺时针赋值，当前角没有数值则与对角取值相同
  • 圆角半径最大值为50%，超过之后还是显示50%

阴影

  • box-shadow: X轴偏移量 Y轴偏移量 模糊半径 扩散半径 颜色 内外阴影;
  • X轴偏移量和Y偏移量必须书写
  • 默认是外阴影，内阴影需要添加insert

CSS书写顺序

  1. 盒子模型属性
  2. 文字样式
  3. 圆角、阴影等修饰属性

标准流

  • 标准流也称文档流，指的是标签在页面中默认的排布规则
    • 例如，块元素独占一行
    • 行内元素可以一行显示多个

浮动

  • 让块级元素水平排列
属性值 效果
left 左对齐
right 右对齐
  • 浮动后的盒子
    • 顶对齐
    • 具备行内块的特点
    • 脱标，不占用标准流的位置
    • 父级宽度不够，浮动的子级会换行

清除浮动

  • 浮动元素会脱标，如果父级没有高度，子级无法撑开父级高度（可能导致页面布局错乱）
  • 解决：清除浮动（清除浮动带来的影响）
    • 方法一：额外标签法
      • 在父元素的最后添加一个块级元素，设置CSS属性clear: both;
      • 一般设置该块级元素为class = "clearfix"
    • 方法二：单伪元素法
    • 方法三：双伪元素法（推荐）
    • 方法四：overflow
      • 父元素添加CSS属性overflow: hidden;
  • 浮动的本质作用是实现图文混排效果（即文字环绕图片）

Flex布局

  • Flex布局又称弹性布局，是浏览器提倡的布局模型，非常适合结构化布局，提供强大的空间分布和对齐能力
  • Flex模型不会产生浮动布局中的脱标现象，布局网页更简单、更灵活

Flex组成

  • 设置方式：给父元素设置display: flex;，子元素可以自动挤压或拉伸
  • 组成部分：弹性容器、弹性布局、主轴：默认在水平方向、侧轴/交叉轴：默认在垂直方向

常用属性

描述 属性
创建flex容器 display: flex
主轴对齐方式 justify-content
侧轴对齐方式 align-items
某个弹性盒子侧轴对齐方式 align-self
修改主轴方向 flex-direction
弹性伸缩比 flex
弹性盒子换行 flex-wrap
行对齐方式 align-cont
